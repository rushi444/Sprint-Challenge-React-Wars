    Font Star Jedi for site

    You can connect the font Star Jedi to your site, without downloading it from our server. All you need to do - is to follow the instructions below:

    1. Choose the most suitable option to install the font and add this code to your site (paste it immediately after the opening tag <head>):

      • Standard
      • @import

        <link href="http://allfont.net/allfont.css?fonts=star-jedi" rel="stylesheet" type="text/css" />

        @import url(http://allfont.net/allfont.css?fonts=star-jedi);

    2. Activate the font for text in your CSS:

        font-family: 'Star Jedi', arial;

    Sample source code:

    <html>
  <head>
    <link href="http://allfont.net/allfont.css?fonts=star-jedi" rel="stylesheet" type="text/css" />
    <style>
      body {
        font-family: 'Star Jedi', arial;
        font-size: 48px;
        text-shadow: 4px 4px 4px #aaa;
      }
    </style>
  </head>
  <body>
    <div>Making the Web Beautiful with Star Jedi!</div>
  </body>
</html>
